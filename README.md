# Project-Based Excel for Data Analysts

## About This Repository

This repository documents my journey of learning **Excel for Data Analysis** through project-based learning.

Instead of following long video courses or getting stuck in "tutorial hell," I wanted a learning approach that emphasizes **building first and learning through practice**. To make that possible, I designed these projects with the help of AI, mainly experimenting with **Skywork AI** as a learning partner.

Rather than asking AI to teach me Excel directly, I prompted it to generate a structured curriculum inspired by the project-based philosophy used in intensive coding schools. The idea is simple:

> **Learn by doing. Solve real problems first, research what you need, make mistakes, and improve through practice.**

The projects in this repository were created specifically for that purpose. They are not copied from an existing course but are designed as progressively more challenging exercises that simulate the work of a data analyst.

## Learning Philosophy

The curriculum follows a project-driven approach:

* No passive watching.
* No memorizing theory before practice.
* Every concept is learned by solving a real task.
* Each project builds on skills learned in previous ones.
* Documentation and self-review are part of the learning process.

The goal is not only to learn Excel but also to develop the mindset of solving business problems with data.

## Curriculum Overview

### Phase 1 — Excel Fundamentals

Phase 1 builds mechanical confidence. You will learn how Excel stores numbers, how formulas reference
cells, how to aggregate data, how to branch based on conditions, and how to build your first chart-driven
mini-dashboard.

By the end of this phase, the objective is to comfortably manipulate datasets, write reliable formulas, and create clear visual summaries.
