# Phase 1 / ex01 — SUM and AVERAGE
## Objective
Use core aggregation formulas to total, average, and compare revenue across products and months.
## The Tools
=SUM(range), =AVERAGE(range), =SUMPRODUCT(array1,array2)
## Tasks
1. Calculate total Q1 revenue for each product category.
2. Calculate the average monthly revenue for Electronics.
3. Find the total revenue for February across all categories.
4. Determine which month had the highest total revenue.
5. Calculate the weighted average revenue per unit across all categories using total Q1 revenue and Units Sold.
