# Phase 1 / ex00 — MIN and MAX

## Objective
Learn how to identify the highest and lowest values in a range and use those results to answer business questions quickly.
---
## The Tools
=MAX(range), =MIN(range)
---
## Tasks
1. Find the highest sales value in Week 1.
2. Find the lowest sales value in Week 2.
3. Find Charlie's lowest weekly result across the four weeks.
4. Find the absolute maximum value across the entire dataset.
5. Calculate the Week 3 range, defined as maximum minus minimum.
